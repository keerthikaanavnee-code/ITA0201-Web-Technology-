📊 GET vs POST Comparison

Feature GET Method POST Method
Purpose Search Data Submit Data
Data Location Visible in URL Sent in Request Body
Security Less Secure More Secure
Bookmark Yes No
Used In Train Search Ticket Booking
URL Changes Yes No

📝 Observation


When the Search Train form is submitted using the GET method, the entered values are appended to the browser URL as query parameters. This makes the data visible in the address bar and allows the page to be bookmarked.


When the Book Ticket form is submitted using the POST method, passenger details are sent in the request body. Therefore, the URL remains unchanged, making POST more suitable for sending sensitive information.
